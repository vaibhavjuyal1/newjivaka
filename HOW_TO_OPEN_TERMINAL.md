# How to Open Terminal in VS Code/Cursor

## Method 1: Keyboard Shortcut

**Press:** `Ctrl + ` (Control + Backtick)

**Where is the backtick key?**
- Look above the **Tab** key
- Same key as the **tilde (~)** symbol
- Usually on the **top-left** of your keyboard, left of the number 1

**Visual guide:**

## Method 2: Menu

1. Click **"Terminal"** menu at the top
2. Click **"New Terminal"**

## Method 3: Command Palette

1. Press `Ctrl + Shift + P` (Control + Shift + P)
2. Type: "Terminal: Create New Terminal"
3. Press Enter

## Method 4: Right-Click

1. Right-click on any file in the left sidebar
2. Select **"Open in Integrated Terminal"**

STEP 1: DEPLOY BACKEND (5 minutes)

1. Open your web browser
2. Go to: https://vercel.com
3. Click "Sign Up" (or "Log In" if you have account)
   - Use Google/GitHub/Email (all free)
4. After login, click "Add New..." button (top right)
5. Click "Project"
6. You'll see 2 options:
   
   OPTION A - Upload Folder (Easiest):
   - Drag and drop your D:\jivaka\backend folder
   - OR click "Browse" and select D:\jivaka\backend folder
   
   OPTION B - Connect GitHub:
   - Click "Import Git Repository"
   - Connect GitHub account (if you have one)

7. After uploading/connecting:
   - Root Directory: Type "backend" (or leave blank if it auto-detects)
   - Build Command: Leave empty
   - Output Directory: Leave empty
   - Install Command: Type "npm install"
   - Start Command: Type "node server.js"

8. Click "Environment Variables" section
9. Click "Add" button
10. Add these 2 variables (one at a time):
    
    Variable 1:
    - Name: NODE_ENV
    - Value: production
    - Click "Add"
    
    Variable 2:
    - Name: PORT
    - Value: 5000
    - Click "Add"

11. Scroll down, click "Deploy" button
12. Wait 2-3 minutes (watch the progress bar)
13. When done, you'll see a green "Deployment successful" message
14. Copy your backend URL (looks like: https://jivaka-backend-xxx.vercel.app)
    - Click on the URL to copy it
    - SAVE THIS URL - You'll need it for Step 2!

STEP 2: DEPLOY FRONTEND (5 minutes)

1. Still on Vercel website
2. Click "Add New..." button again (top right)
3. Click "Project"
4. Upload/Connect frontend:
   
   OPTION A - Upload Folder (Easiest):
   - Drag and drop your D:\jivaka\frontend folder
   - OR click "Browse" and select D:\jivaka\frontend folder
   
   OPTION B - Connect GitHub:
   - Click "Import Git Repository"
   - Select frontend folder

5. After uploading:
   - Root Directory: Type "frontend"
   - Framework Preset: Select "Create React App"
   - Build Command: Type "npm run build"
   - Output Directory: Type "build"
   - Install Command: Type "npm install"

6. Click "Environment Variables" section
7. Click "Add" button
8. Add this variable:
   
   - Name: REACT_APP_API_URL
   - Value: https://YOUR-BACKEND-URL/api
     (Replace YOUR-BACKEND-URL with the URL you copied in Step 1!)
     Example: https://jivaka-backend-xxx.vercel.app/api
   - Click "Add"

9. Scroll down, click "Deploy" button
10. Wait 2-3 minutes
11. When done, you'll see your frontend URL (like: https://jivaka.vercel.app)

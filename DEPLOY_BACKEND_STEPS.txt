═══════════════════════════════════════════════════════════════
  STEP-BY-STEP: Deploy Backend
═══════════════════════════════════════════════════════════════

1. Click "Add New..." button (top right)
2. Click "Project"

3. UPLOAD BACKEND:
   - Option A: Drag D:\jivaka\backend folder into Vercel
   - Option B: Click "Browse" → Select D:\jivaka\backend

4. FILL THESE BOXES:
   - Root Directory: backend
   - Install Command: npm install
   - Start Command: node server.js
   - Build Command: (leave empty)
   - Output Directory: (leave empty)

5. SCROLL DOWN to "Environment Variables"
   - Click "Add" button
   - Name: NODE_ENV
   - Value: production
   - Click "Add" again
   - Name: PORT
   - Value: 5000

6. Click "Deploy" button (big blue button)

7. WAIT 2-3 minutes

8. When you see green checkmark ✅, copy the URL!
